### Basic API with other MOdel

#### Requirement 
- net/http
- httprouter (github.com/julienschmidt/httprouter)
- controllers (owned by self)

#### Pattern URI

| HTTP        | Pattern          | 
| ------------- |:-------------:| 
| GET     | /user/id | 
| POST      | /user |
| DELETE    | /user/id|


#### HOW 
`go run main.go`

Listen port `8181`, Open in browser `http://localhost:8181`
